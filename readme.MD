
# E-Commerce with Java

A comprehensive e-commerce platform built with Java, featuring product management, shopping cart functionality, and order processing.

## Features

- Product catalog and search
- Shopping cart management
- User authentication
- Order management
- Payment processing
- Inventory tracking

## Getting Started

### Prerequisites

- Java 11 or higher
- Maven 3.6+
- Database (MySQL/PostgreSQL)

### Installation

1. Clone the repository
2. Configure database connection in `application.properties`
3. Run `mvn clean install`
4. Start the application with `mvn spring-boot:run`
